## 📌Today I Learned
### useRef() VS useState()
- 둘 다 React 컴포넌트 내부에서 상태나 데이터를 관리할 때 사용됨  
- useRef()
    - 값이 바뀌어도 리렌더링 되지 않음
    - 렌더링이 돼도 값이 바뀌지 않음
    - DOM 참조하거나 렌더링과 무관한 값 저장할 때 쓰임
- useState()
    - 값이 바뀌면 리렌더링 발생
    - 렌더링이 돼도 값이 바뀌지 않음
    - UI 상태나 사용자 인터랙션 기반 상태에 쓰임
